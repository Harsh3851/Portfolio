# 🚀 Harsh Shukla — Portfolio

A modern, responsive personal portfolio website showcasing my projects, technical skills, and professional background.

🔗 **Live Site**: [harsh3851.github.io/Portfolio](https://harsh3851.github.io/Portfolio/)

## ✨ Features

- **Animated Hero Section** — Typed.js-powered dynamic role titles with floating icon animations
- **Project Showcase** — Hover-interactive project cards with overlay transitions and direct GitHub/demo links
- **Skills Grid** — Categorized tech stack displayed in glassmorphic cards with brand-colored icons
- **Contact Footer** — Dark-themed footer with quick links, social icons, and direct email/resume access
- **Responsive Design** — Fully responsive across desktop, tablet, and mobile viewports

| Layer         | Technology                                                      |

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- npm (comes with Node.js)

### Local Development

```bash
# Clone the repository
git clone https://github.com/Harsh3851/Portfolio.git
cd Portfolio

# Install dependencies
npm install

# Start the local dev server
npm run start
```

The site will be available at **http://localhost:3000**.

### Deployment

The site is deployed automatically via **GitHub Pages** from the `main` branch.  
Any push to `main` will trigger a new deployment at:  
👉 [https://harsh3851.github.io/Portfolio/](https://harsh3851.github.io/Portfolio/)

## 📸 Sections

### Hero
Animated introduction with typed role titles, floating icons, and a profile image with a grayscale-to-color animation.

### Projects
Three featured project cards with hover-reveal overlays, numbered badges, and links to GitHub repos and live demos.

### Skills
Eight categorized skill cards (Languages, Frameworks, Web, Backend & APIs, AI Tools, Tools, Testing, Methodologies) displayed in a 2-column grid with brand-colored icons and a decorative blob gradient.

### Contact / Footer
Dark-themed footer with personal branding, quick navigation links, external resources (GitHub, Resume, LinkedIn), and direct email contact.
